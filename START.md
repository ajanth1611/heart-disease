# Run the app

Use the bundled Python environment in `venv37`.

Windows Command Prompt:

```bat
cd "c:\Users\user\OneDrive\Desktop\heart disease"
run_app.bat
```

PowerShell:

```powershell
cd "c:\Users\user\OneDrive\Desktop\heart disease"
.\run_app.ps1
```

If you prefer not to use the script, run directly:

```powershell
cd "c:\Users\user\OneDrive\Desktop\heart disease"
.\venv37\Scripts\python.exe -m streamlit run pages\1_Prediction.py
```
